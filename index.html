A Redmine plugin which makes configuring your own Git hosting easy ;)

This plugin allows straightforward management of Gitolite within Redmine.

It provides these functionalities :

  • SSH Public Keys Management
  • Automatic Repository Creation
  • Repository Deletion
  • Deployment Credentials
  • Post-Receive URLs
  • Automatic Mirror Updates
  • Git Smart HTTP (with vulnerability patch)
  • Git Daemon
  • Caching Options
  • (new) Git mailing lists
  • (new) Sidekiq async jobs
  • (new) Default branch selection
  • (new) Automatic Repository Initialization
  • (new) Git Revision Download (from Git Revision Download)
  • (new) README Preview (from README at Repositories)
  • (new) Git Config Keys Management
  • (new) Improved Repository Statistics
  • (new) Github Issues Sync
  • (new) Browse Archived Repositories

You can follow announcements here or take a look at the roadmap.
