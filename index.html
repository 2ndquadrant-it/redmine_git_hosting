    • Redmine Git Hosting

    A Redmine plugin which makes configuring your own Git hosting easy :)

    Project maintained by jbox-web Hosted on GitHub Pages — Theme by mattgraham

    This plugin allows straightforward management of Gitolite within Redmine.

    It provides these functionalities :

    • SSH Public Keys Management
    • Deployment Credentials
    • Post-Receive URLs
    • Automatic Mirror Updates
    • Git Smart HTTP
    • Git Daemon
    • Caching Options
    • Git mailing lists
    • Sidekiq async jobs

    You can follow announcements here or take a look at the roadmap.
